Перейдите в Терминал и запустите контейнер с брокером Kafka с помощью команды:
docker run -d --name container_name apache/kafka:latest

Для взаимодействия с Kafka перейдите в интерактивную оболочку внутри контейнера:
docker exec --workdir /opt/kafka/ -it --user=root container_name sh

================================================================================

Apache Kafka предоставляет пользователям ряд утилит командной строки (CLI - Command Line Interface), удобных для администрирования кластера. Эти утилиты позволяют выполнять простейшие операции, например:

* создать топик
* создать продьюсера
* создать консьюмера
* запускать/останавливать брокер

По сути, каждая утилита – это небольшая программа на Java.

Все утилиты лежат в одной директории - bin и имеют расширение .sh.
