        Если кластер был развёрнут с помощью - kubeadm

- kubeadm certs check-expiration                 - просмотр всех сертификатов
- kubectl -n kube-system logs etcd-controlplane  - просмотр всех логов

- docker ps | grep etcd            - список контейнеров etcd
- docker logs some_container_id    - логи контейнера по его ID
