
                    Имя загружаемого на DockerHub образа должно выглядеть так: username/imagename:tag. Разберём его элементы:

    username — имя пользователя на DockerHub.
    imagename — произвольное имя образа. Оно не может быть короче 22 и длиннее 255255 символов, должно содержать только строчные буквы, цифры, дефисы (-) и нижние подчёркивания (_). При этом начинать имя образа с дефиса нельзя.
    tag — необязательная, но полезная часть имени. Применение тегов даёт возможность загружать в облако различные версии одного и того же образа. Например, на DockerHub существует пользователь billglasses. Его репозиторий может состоять из таких образов:
      
        billglasses/gates:v1.1985;
        billglasses/gates:v2.1987;
        billglasses/gates:v2.11.1989;
        billglasses/gates:v3.1990.

  Обращаться к образу лучше через указание тега: docker pull billglasses/gates:v2.11.1989. 
Допускается обращение и без тега: billglasses/gates. 

  Тогда устанавливается последняя версия этого образа — такая команда аналогична billglasses/gates:latest. 
Но хорошей практикой всё-таки считается указание конкретной версии.



                                                    Загрузка образа на DockerHub

  Допустим, пользователь billglasses решил загрузить в репозиторий определённую версию образа gates. 

Для этого пользователь должен:

    Локально создать образ с нужным названием и тегом.

 docker build -t billglasses/gates:v2.11.1989 .
  

Авторизоваться через консоль.

 docker login
 # а можно сразу указать имя пользователя
 docker login -u billglasses
  

Загрузить образ на DockerHub.

 docker push billglasses/gates:v2.11.1989
  

  Если понадобится изменить образ, нужно локально внести изменения в Dockerfile, 
пересобрать образ и выполнить команду docker push <imagename> — образ обновится на DockerHub. 
  Чтобы обновить образ, который лежит на сервере или на вашем локальном компьютере, 
обновите образ на DockerHub и выполните на сервере или компьютере команду docker pull <imagename>. 
Образ скачается на сервер — останется только запустить контейнер.
