Общий вид команды:
команда   опции   аргументы
ls   --all --human-readable -l   путь
ls   -a -h -l   путь
ls   -ahl   путь   или   ls   -lah   путь

ls -al                     -  все файлы и папки, включая скрытые + информация о них
ls -alt                    -  все файлы и папки, включая скрытые + информация о них + сортировка по дате

man команда  —  справка о команде (для выхода — q)
clear        —  очистить экран терминала
exit         —  закрыть терминал
file         -  узнать информацию о файле (расширение и т.п.)
   (Важно отметить, что для работы с такими специальными файлами,
    как файлы из директорий /dev и /proc,
    следует использовать команду file -s)

Символы путей:
.     - текущая директория
..    - директория на уровень выше
../.. - 2 директории на уровень выше
~     - домашняя директория
*     - любое количество любых символов
?     - один любой символ
-     - вернуться в предыдущую папку

ПАПКИ
mkdir «путь до папки»                       - создать папку
sudo mkdir «путь до папки»                  - создать папку с правами администратора
mkdir -p - «путь до папки»                  - создать папку и её «родителей»
touch «путь до файла»                       - создать файл
sudo chown root:serega fileName.txt         - поменять права доступа файла
sudo chown -R root:serega dirName           - поменять права доступа папки
sudo chmod                                  - поменять модификацию файла или папки
           4                                : доступ только для чтения
           6                                : доступ для чтения и записи
           7                                : обсалютный доступ (лучше никогда не делать)
sudo chmod суперUser, группа, обычный user
sudo chmod 664 fileName                     : -rw-rw-r--

rm «путь до файла»       - удалить файл
rm -r «путь до папки»    - удалить папку
rm -rf «путь до папки»   — удалить всю папку (или весь путь), без вопросов
rm dirName/*             - удалить все файлы(не папки) в папке dirName. (надо находиться выше)
rm dirName/*.txt         - удалить все файлы .txt в папке dirName. (надо находиться выше)

cp «путь до файла1» «путь до файла2»    - скопировать файл1 в файл2
cp -r «путь до папки1» «путь до папки2» - скопировать папку1 в папку2
mv «путь1» «путь2»                      - переместить папку или файл
mv fileName newFileName                 - переименование файла

                                        echo

- echo "Ау!"                  - вывели в окно терминала строку "Ау!"
- echo "text" > file_name     - строка text будет записана в файл filename. Если файла с таким названием нет, то он будет создан.

                              Как и с другими командами, путь до файла можно указать как абсолютный, так и относительный:

                              # абсолютный путь
                              echo "Я отличный программист" > /secrets/secret.txt

                              # относительный путь
                              echo "готовность_к_обучению: 100.00" > ../settings/my_settings.txt

                                        cat

  Чтобы поменять текст в одном файле на текст из другого, cat применяют так — cat [источник] > [назначение]

  # вывести содержимое файла a.txt в окно терминала
  cat a.txt
  AAA
  # в файле a.txt хранится строка AAA

  # вывести содержимое файла b.txt в окно терминала
  cat b.txt
  BBB
  # в файле b.txt хранится строка BBB

  # заменить содержимое файла b.txt на содержимое файла a.txt
  cat a.txt > b.txt

  # данные в файле b.txt перезаписаны, содержимое файлов идентично
  cat b.txt
  AAA
  cat a.txt
  AAA


  Если нужно скопировать содержимое одного файла и добавить в конец другого, требуется использовать сдвоенные скобки >>

# снова сохранить строку ВВВ в b.txt
echo "BBB" > b.txt

# скопировать содержимое a.txt в b.txt, не перезаписывая данные
cat a.txt >> b.txt

# вывести b.txt
cat b.txt
BBB
AAA

    Можно сохранить в файл данные сразу из нескольких файлов

# соединить содержимое 1.txt и 3.txt и сохранить в конец b.txt
cat 1.txt 3.txt >> b.txt

cat b.txt
BBB
AAA
Раз
Три

# соединить содержимое 2.txt и 1.txt и сохранить в b.txt, перезаписав данные
cat 2.txt 1.txt > b.txt

cat b.txt
Два
Раз

                  СКАЧИВАНИЕ ИЗ ИНТЕРНЕТА

wget:    ссылка
скачать файл по ссылке и сохранить в
текущей директории

wget -P:     путь_до_директории ссылка
скачать файл по ссылке и сохранить в
директории заданной путем

wget -O:    путь_до_файла ссылка
скачать файл по ссылке и сохранить под
указанным именем

wget -c:    ссылка
докачать файл по ссылке в случае
обрыва связи

wget —spider:    ссылка
проверить доступность файла по
ссылке

wget -i:    текстовый_файл
скачать несколько файлов по ссылкам
из текстового файла

wget -r -l:    глубина ссылка
рекурсивное скачивание файлов по
ссылке на указанную глубину
(по умолчанию глубина 5)

wget -r -A:    тип,тип,...,тип ссылка
рекурсивное скачивание файлов только
определенного типа (типов)
