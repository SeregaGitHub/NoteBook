- --kubelet-insecure-tls    - быстрый обходной вариант подключения томов сертификатов metrics-server

Параметр --kubelet-insecure-tls пропускает проверку сертификатов CA Kubelet. 
Это может быть полезно, если используются самозаверяющие сертификаты, которые не могут быть проверены сервером метрик. 

Важно: отключение проверки TLS может быть не рекомендуется для производства — 
в документации Metrics Server указано, что этот параметр не является решением для производства.
