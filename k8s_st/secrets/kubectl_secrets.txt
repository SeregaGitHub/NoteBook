

- kubectl apply -f secret_file_name.yaml    - создать секрет из файла манифеста
- kubectl get secrets                       - получить список секретов
- kubectl describe secret secret_name       - получить описание секрета
- kubectl get secret secret_name -o yaml    - получить подробное описание секрета (с хэшированными значениями)
- kubectl delete secret secret_name         - удалить секрет

------------------------------------------------------------------------------

- echo -n "some_text" | base64              - закодирует текст для секрета
- echo -n "hash_text" | base64 --decode     - раскодирует хэш из секрета
