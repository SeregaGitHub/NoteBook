                                    Создание образа VM

* Перейдите в сервис Compute Cloud → Диски.
* Нажмите на три точки напротив загрузочного диска ВМ. В появившемся меню нажмите Создать образ.
* Оставьте поля по умолчанию и нажмите Создать.

                                Настройка прав доступа к образу

* Перейдите в раздел Образы и нажмите на созданный образ. Затем перейдите в раздел Права доступа и нажмите Назначить роли.
* В открывшемся окне в списке выберите группу All authenticated users — доступ для всех аутентифицированных пользователей.
* Нажмите Добавить роль и выберите compute.images.user.
