cat (имя файла) — вывести содержимое файла в терминале 

less (имя файла) — открыть файл на чтение:
q — выход 
/ - поиск
g — в начало
G — в конец

nano (имя файла) — редактировать файл
Ctrl + X — выход

     ПЕРЕНАПРАВЛЕНИЕ В ФАЙЛ

        Программа<файл
      брать stdin из файла

        Программа>файл
     выводить stdout в файл

        Программа>>файл
 аналогично, но с дозаписью в файл

        Программа2>файл
     выводить stderr в файл

        Программа2>>файл
 аналогично, но с дозаписью в файл



           КОНВЕЙЕР (pipe)
     Прогр1|Прогр2 |...| ПрогрN

 Передать  stdout Прогр1 в качестве  stdin для  Прогр2 и т. д.

 cat new_names.txt | ./iteracter.py | less
 открыть файл  new_names.txt | программа ./iteracter.py обработает файл | less откроет файл
